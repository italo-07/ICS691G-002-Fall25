## Practices and Challenges of Using GitHub Copilot: An Empirical Study

<em>The 35th International Conference on Software Engineering and Knowledge Engineering (SEKE)</em> <br />

The growth of AI has created a surge in interest and adoption, one of them being GitHub Copilot. However, a recent study suggests that there are critical considerations on the security, intellectual property, and code quality of GitHub Copilot. Additionally, these researchers have found that the main programming languages used with Copilot are Javascript and Python, the main IDE used is VSCode, and the leading function implemented with this tool is data processing.
